# 📚 Système de Gestion de Bibliothèque Numérique (Java EE)

Projet réalisé dans le cadre du module **Génie Informatique / RSI**.

### 🚀 Fonctionnalités
- **Dashboard Admin** : Gestion complète (CRUD) des ouvrages.
- **Upload Dynamique** : Gestion des vignettes (couvertures) via `MultipartConfig`.
- **Lecture PDF** : Liaison directe entre la fiche ouvrage et le fichier local via un chemin dynamique.
- **Sécurité** : Gestion des sessions et restrictions d'accès aux servlets d'administration.

### 🛠️ Technologies
- **Backend** : Java EE (Servlets, JSP, JSTL)
- **Base de données** : MySQL / PostgreSQL
- **Architecture** : MVC (Model-View-Controller)
- **Frontend** : CSS3 Custom (Style Rose/Gold moderne)
